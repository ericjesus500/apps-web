¿De qué trata esta web?

Apuntes, Manuales, Presentaciones y otros materiales relacionados con las materiales que imparto o he impartido en el pasado, son la base de los contenidos de este sitio.

Puesto que, como profesor, he utilizado y utilizo numerosos recursos disponibles en Internet, hace años que decidí contribuir yo también con mis propios materiales.

Qué temas se cubren

He agrupado los contenidos en base a estos temas:

  • Bases de Datos. Dentro de este apartado se presentan materiales sobre las asignaturas que impartido, o he impartido, de Administración de Sistemas Informáticos en Red (ASIR)y cursos y ponencias sobre bases de datos. Se divide este apartado en:
    • Gestión de Bases de Datos. Donde, sobre todo, hay materiales de los módulos con el mismo nombre impartidos dentro del ciclo de ASIR. Entre los temas que tratan los materiales están los lenguajes SQL y PL/SQL, Diseño de bases de datos Relacionales, Bases del Modelo Relacional
    • Administración de Bases de Datos. Con apuntes, sobre todo, de administración de Sistemas Oracle Database y otros materiales relacionados
    • Modelo Entidad/Relación. Incluye detalles y ejercicios sobre el modelo y una aplicación web (en fase beta) de diseño de esquemas Entidad/Relación (erMaker)
    • Álgebra Relacional. Con ejercicios sobre lenguaje de bases de datos.
  • Creación y desarrollo web. Organizados en estos apartados:
    • Lenguajes de Marcas y Sistemas de Gestión de Información. Contiene apuntes y PowerPoints relacionados con este módulo del ciclo de ASIR.
    • Implantación de Aplicaciones Web, Contiene libros, apuntes y PowerPoints relacionados con este módulo del ciclo de ASIR.
    • HTML 5. Contiene material de aprendizaje del lenguaje fundamental de creación de aplicaciones web. Incluye material sobre HTML, CSS y JavaScript
    • XML. Apuntes y PowerPoints sobre XML.
    • PHP. Apuntes y PowerPoints sobre el lenguaje PHP.
    • Libro de Implantación de Aplicaciones Web. Referido al libro publicado por la editorial Garceta referido a este tema.
  • Programación de Aplicaciones, en este caso con manuales y apuntes relativos a temas relacionados con lenguajes de programación de aplicaciones. Se ha dividido el apartado en:
    • Fundamentos de Programación. Apuntes y otros materiales sobre el módulo del mismo nombre impartido en el ciclo deAdministración de Sistemas Informáticos
    • Lenguaje Java
    • Lenguaje C
    • Lenguaje C++
  • Informática Básica y Ofimática.
  • Imagen Digital.

Puede que haya erratas y fallos (desgraciadamente muy a menudo) en el material que publico, de ahí que me sea muy enriquecedor los comentarios o mensajes que me hacéis llegar. Muchas gracias por ello.

Transformaciones mediante CSS

[18.1] transformaciones

Por transformaciones en CSS se entiende las operaciones realizadas mediante la propiedad transform, la cual permite las operaciones típicas vectoriales sobre un elemento HTML. En realidad hay dos tipos de transformaciones, las transformaciones 2D, que sólo utilizan los ejes horizontal y vertical para transformar, lo que implica que sólo realiza transformaciones en el plano; y las transformaciones 3D en las que interviene el eje Z causando la ilusión de que el elemento tenga profundidad.

[18.2] transformaciones 2D

A la propiedad transform se la acompaña de una función que es la responsable de realmente el elemento. Esa función puede ser:

  • scale(x,y). Realiza un escalado sobre el elemento. El primer valor indica el factor de escalado horizontal (1 es el 100% y .5 escala al 50%) y el segundo el escalado vertical.
  • scaleX(x). Escala el elemento sólo en horizontal.
  • scaleY(y). Escala el elemento sólo en vertical.
  • rotate(ángulo). Rota el elemento el ángulo indicado (por ejemplo 45deg)
  • rotate(ángulo). Rota el elemento el número de grados indicados
  • skewX(angle). Estira el elemento en el eje horizontal
  • skewY(angle). Estira el elemento en el eje vertical
  • skew(ánguloX, ánguloY). Estira el elemento usando un ángulo en horizontal y otro en vertical.

Los navegadores actuales soportan la propiedad transform, pero los de tipo webkit (Chrome y Safari) imponen su versión llamada –webkit-transform. También Internet Explorer usa su propia versión (aunque ahora ya usa la estándar) y hasta hace poco hacía lo mismo Firefox.

<!doctype html>
<html lang=”es”>
<head>
	<meta charset=”UTF-8”>
	<title>Document</title>
	<style>
		div{
			width:100px;
			height:100px;
			float:left;
			margin:5px;
			background-color: yellow;
			border:1px solid black;
		}
		#d2{
			transform: scale(1,.5);
		}
		#d3{
			transform:rotate(45deg);
		}
 
		#d4{
			transform:skewX(-45deg);
		}
		#d5{
			transform:skewY(-45deg);
		}
	</style>
</head>
<body>
	<div id=”d1”>Texto1</div>
	<div id=”d2”>Texto2</div>
	<div id=”d3”>Texto3</div>
	<div id=”d4”>Texto4</div>
	<div id=”d5”>Texto5</div>
</body>
</html>

[18.3] transformaciones 3D

En este caso hay que tener en cuenta que permiten distinguir el eje sobre el que se aplican. Las posibles funciones que podemos utilizar en la propiedad transform para conseguir transformaciones 3D son:

  • translate3D(x,y,x). Traslada el elemento pudiendo indicar desplazamientos en los 3 ejes a la vez.
  • translateX(x) Traslada el elemento en el eje X.
  • translateY(y) Traslada el elemento en el eje Y.
  • translateZ(y) Traslada el elemento en el eje Z.
  • scale3D(x,y,x). Escala el elemento pudiendo diferenciar escalas en las tres dimensiones del elemento
  • escaleX(X) Escala el elemento en horizontal
  • escaleY(Y) Escala el elemento en vertical
  • escaleZ(Z) Escala el elemento en el eje Z
  • rotate3D(x,y,x). Rota el elemento pudiendo indicar una rotación distinta en cada eje
  • rotateX(X) Rota el elemento usando el eje X de coordenadas. Un elemento normal rotado más de 90 grados, hará un efecto espejo. Rotando 90 grados exactos, el elemento queda invisible al quedar absolutamente horizontal a nuestra vista.
  • rotateY(Y) Rota el elemento usando el eje Y de coordenadas. Como en la rotación anterior, un elemento normal rotado más de 90 grados, hará un efecto espejo. También justo con 90 grados el elemento queda invisible al ser absolutamente vertical a nuestra vista.
  • escaleZ(Z) Escala el elemento en el eje Z, es la rotación 2D normal por lo tanto.
  • perspective(n). Define un punto de perspectiva para las transformaciones 3D. Es equivalente a utilizar la propiedad perspective: n.

Hay otras propiedades relacionadas con las transformaciones 3D:

  • transform-style. Si indicamos a esta propiedad el valor preserve-3d, si este objeto contiene a su vez objetos transformados en 3D, estas transformaciones se mantendrán. La otra opción es indicar el valor flat.
  • transform-origin. Permite trasladar el objeto transformado a otra posición, de la cual se indica valores de eje X e Y.

    Ejemplo transform-origin:200px 100px;

  • perspective. Posición de la perspectiva de la persona que observa la escena.
  • perspective-origin. Permite indicar donde está el eje de coordenadas X,Y. Por defecto vale 50% 50% (justo en el centro). Admite indicar las palabras left, center o right para el eje horizontal y top, center o bottom para el vertical

A continuación se muestra un ejemplo con transformaciones 3D. Se trata de cuatro capas interiores a un contenedor. Las cuatro capas se colocan llenando dicho contenedor (usando posicionamiento absoluto), después se rotan en perspectiva formando las tapas de un cubo tridimensional visto en perspectiva.

<!doctype html>
<html lang=”es”>
<head>
    <meta charset=”UTF-8”> <title>Ejemplos 3D</title>
    <style>
        #contenedor{
            position:fixed;
            left:400px;
            top:400px;
            width:400px;
            height:400px;
            border:4px solid yellow;
            transform-style: preserve-3d;
            perspective-origin: center center;
            perspective:500px;
 
        }
        #contenedor div{
            position:absolute;
            width:100%;
            height:100%;
            left:0;
            top:0;
            font-size:350px;
            text-align: center;
            color:white;
        }
        #capa1{
            background:linear-gradient(black, red);
            transform:rotateX(90deg) translateZ(200px);
 
        }
        #capa2{
            background:
					linear-gradient(-90deg,black, blue);
				transform:rotateY(90deg) translateZ(200px);
 
        }
        #capa3{
            background:
					linear-gradient(0deg,black, orange);
 
           transform:rotateX(-90deg) translateZ(200px);
 
        }        
#capa4{
            background:
					linear-gradient(90deg,black, green);
           transform:rotateY(-90deg) translateZ(200px);
 
        }
    </style>
</head>
 
<body>
<div id=”contenedor”>
   <div id=”capa1”>1</div>
   <div id=”capa2”>2</div>
   <div id=”capa3”>3</div>
   <div id=”capa4”>4</div>
</div>
</body>
</html>
